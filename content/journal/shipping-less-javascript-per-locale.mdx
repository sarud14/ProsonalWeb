## The hidden cost of locales

Client-side i18n quietly ships every translation a user might need. Across four markets that was a lot of bytes doing nothing on most page loads.

## Resolve on the server

Moving locale resolution into edge middleware meant each request rendered with exactly the strings it needed. The client stopped paying for markets it would never show.
